# Regression projects with datasets
This is a repository containing various projects related to regression, using different statistical tools and approaches (frequentist and Bayesian).


The first project is based on obtaining a dataset that gathers information from high school students, finding an optimal AIC from all the survey options, and assessing its predictive capacity.

The second project uses a Bayesian approach to analyze spatial and temporal models and their descriptive capacity in order to analyze breast cancer rates across the states of the Mexican Republic. The implementation of the models was based on the Knorr-Held model.
